Welcome to a website

This page demonstrates how you can use a Vaadin web application embedded as part of you website. On the sidebar you can see a Vaadin application embedded as a web component. The actual application is deployed in another domain at vaadin-cors-sample.fly.dev. Find the source code for this demo at github.com/samie/vaadin-cors-sample and learn more about Vaadin at vaadin.com.
